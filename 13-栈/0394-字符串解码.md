### 🥊 暴力解（朴素对照）

朴素思路：反复找「最内层的 `数字[字母]`」(方括号里只有字母、没有再嵌套)，展开后替换回去，直到没有方括号为止。

```python
import re

class Solution:
    def decodeString(self, s: str) -> str:
        # 每次只匹配最内层的 k[纯字母]，展开后替换；循环到没有方括号
        pattern = re.compile(r'(\d+)\[([A-Za-z]*)\]')
        while '[' in s:
            s = pattern.sub(lambda m: m.group(2) * int(m.group(1)), s)
        return s
```

- 时间复杂度：`O(展开后长度 × 嵌套深度)`，每展开一层都要全串扫描并重建
- 空间复杂度：`O(展开后长度)`
- ⚠️ 每展开一层都重扫整串、反复构造大字符串，嵌套深时很慢。改成「一趟扫描、用栈保存每层现场」即可避免重复展开 → 演进到下方 `O(展开后长度)` 的栈解。

### ⚡ 最优解

```python
class Solution:
    def decodeString(self, s: str) -> str:
        stack = []        # 遇到 '[' 时压入「外层串 + 重复次数」
        cur_str = ''      # 当前这一层正在拼接的字符串
        cur_num = 0       # 当前这一层的重复次数(可能是多位数)
        for ch in s:
            if ch.isdigit():
                cur_num = cur_num * 10 + int(ch)   # 数字可能多位，逐位累加
            elif ch == '[':
                # 进入新的一层：把当前现场(串+次数)压栈，再清空准备接收内层
                stack.append((cur_str, cur_num))
                cur_str, cur_num = '', 0
            elif ch == ']':
                # 本层结束：取出外层现场，把当前串重复 num 次拼回外层
                prev_str, num = stack.pop()
                cur_str = prev_str + num * cur_str
            else:
                cur_str += ch        # 普通字母，直接拼到当前串
        return cur_str
```

- 时间复杂度：`O(展开后总长度)`，每个字符(含展开产物)只处理一次
- 空间复杂度：`O(展开后总长度)`
