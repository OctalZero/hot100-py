| | 时效 | 说明 |
|---|------|------|
| **CodeTop 网站** [codetop.cc](https://codetop.cc/) | ✅ **实时更新** | 带「最近考察时间」，社区用户持续爆料，**看最新趋势用它** |

- **局限**：幸存者偏差（愿分享的人才爆料）、岗位样本不均（后端/客户端多）、记忆偏差、年度波动。
- **结论**：作为「复习方向」很可靠；要最新数据请去**网站**实时筛选（本目录频度基于 2021 快照，仅作基准）。
