# 143. 重排链表

## 📌 题目

给定单链表 `L: L0 → L1 → … → Ln-1 → Ln`，重新排列为 `L0 → Ln → L1 → Ln-1 → L2 → Ln-2 → …`。

要求：**不能只是修改节点的值**，必须实际修改节点的 `next` 指针。**就地完成**，`O(1)` 额外空间。

```
输入：head = [1,2,3,4,5]
输出：[1,5,2,4,3]
```

### 生活中的算法

一排人站成一条长队（1,2,3,4,5），现在要求「最前一个、最后一个、次前一个、次后一个……」重新排队（1,5,2,4,3）。直观做法是：把队伍**从中间断开**，后半段**掉个头**，然后从两队头部**轮流各取一个**缝合起来。

### 思路演进

1. **朴素（违规）**：把链表值读进数组，按下标重排后写回——题目禁止「只改值」，且 `O(n)` 空间。
2. **三步法（最优）**：纯指针操作，`O(1)` 空间。
   - **第 1 步 找中点**：快慢指针，`slow` 停在前半段最后一个节点。
   - **第 2 步 反转后半段**：从 `slow.next` 开始反转，得到逆序的第二段。
   - **第 3 步 交替合并**：两个指针分别走前半段和反转后的后半段，逐个穿插连接。

> 💡 这道题是「链表套路大拼盘」：会它 = 会 [876. 链表的中间结点](https://leetcode.cn/problems/middle-of-the-linked-list/) + [206. 反转链表](../../08-链表/0206-反转链表.md) + 合并。字节爱用它一次考察多项基本功。

## 🐍 Python 代码

### 🥊 暴力解（朴素对照）

把链表节点读进数组，按下标「头取一个、尾取一个」重连——最直观，但用了 `O(n)` 额外空间。

```python
from typing import List, Optional

class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def reorderList(self, head: Optional[ListNode]) -> None:
        if not head or not head.next:
            return

        # 1. 把所有节点读进数组
        nodes: List[ListNode] = []
        cur = head
        while cur:
            nodes.append(cur)
            cur = cur.next

        # 2. 头尾双指针交替重连
        left, right = 0, len(nodes) - 1
        while left < right:
            nodes[left].next = nodes[right]   # 前指针 → 后指针
            left += 1
            if left == right:                 # 奇数个节点时的中点，提前停
                break
            nodes[right].next = nodes[left]   # 后指针 → 新的前指针
            right -= 1
        nodes[left].next = None               # 新链表末尾置空
```

- 时间复杂度：`O(n)`，一次遍历读入 + 一次重连
- 空间复杂度：`O(n)`，用数组存下全部节点
- ⚠️ 题目要求就地 `O(1)` 空间、改指针而非改值。用「快慢指针找中点 + 反转后半段 + 交替缝合」三步法即可做到 `O(1)` 额外空间，见下方最优解。

### ⚡ 最优解

```python
class Solution:
    def reorderList(self, head: Optional[ListNode]) -> None:
        if not head or not head.next:
            return

        # —— 第 1 步：快慢指针找中点（slow 停在前半段末尾）——
        slow = fast = head
        while fast.next and fast.next.next:
            slow = slow.next
            fast = fast.next.next

        # —— 第 2 步：反转后半段 ——
        second = self._reverse(slow.next)
        slow.next = None                  # 前半段断开

        # —— 第 3 步：交替缝合 ——
        first = head
        while second:
            nxt1, nxt2 = first.next, second.next
            first.next = second           # first → second
            second.next = nxt1            # second → 原first下一个
            first, second = nxt1, nxt2

    def _reverse(self, head: ListNode) -> ListNode:
        prev = None
        while head:
            nxt = head.next
            head.next = prev
            prev = head
            head = nxt
        return prev
```
