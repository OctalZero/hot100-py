# 28. 找出字符串中第一个匹配项的下标

## 🛒 人话理解

```mermaid
flowchart TD
    A["每个起点 i"] --> B["haystack[i:i+m] == needle ?"}
    B -- "等" --> C["返回 i ✅"]
    B -- "不等" --> D["i++"]
    D --> E["遍历完 → 返回 -1"]
```

**目标**：在 `haystack` 里找 `needle` 第一次出现的位置。朴素做法是每个起点比较一段子串，O(n·m)。进阶用 KMP（`next` 数组避免主串回退，O(n+m)），适合超大文本高频匹配。

## 🐍 Python 代码

```python
class Solution:
    def strStr(self, haystack: str, needle: str) -> int:
        n, m = len(haystack), len(needle)
        for i in range(n - m + 1):
            if haystack[i:i + m] == needle:
                return i
        return -1
```
