# 102. 二叉树的层序遍历

## 📌 题目

给你二叉树的根节点 `root` ，返回其节点值的 **层序遍历** 。 （即逐层地，从左到右访问所有节点）。

示例：
![](https://assets.leetcode.com/uploads/2021/02/19/tree1.jpg)
```
输入：root = [3,9,20,null,null,15,7]
输出：[[3],[9,20],[15,7]]
```

## 🐍 Python 代码

### 🥊 暴力解（朴素对照）

不用队列，直接 DFS 递归，按节点深度塞进「层号 -> 节点值列表」的字典里，遍历完再按层号顺序输出。

```python
from typing import List, Optional


class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        levels = {}  # 层号 -> 该层节点值列表

        def dfs(node, depth):
            if not node:
                return
            levels.setdefault(depth, []).append(node.val)
            dfs(node.left, depth + 1)
            dfs(node.right, depth + 1)

        dfs(root, 0)
        # 按层号从小到大取出
        return [levels[d] for d in range(len(levels))]
```

- 时间复杂度：`O(n)`，每个节点访问一次
- 空间复杂度：`O(n)`，额外用一个字典存放全部层（外加递归栈）
- ⚠️ 依赖递归栈且层号要先算 `len(levels)` 才能按序输出。用队列天然「先进先出」一层层弹，省去手动按层号重排——演进到下方 BFS 队列解。

### ⚡ 最优解

```python
class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []  # 如果根节点为空，直接返回空列表
        
        result = []  # 用于存储最终的层序遍历结果
        queue = deque([root])  # 初始化队列，将根节点加入队列
        
        while queue:
            level_size = len(queue)  # 记录当前层的节点数
            current_level = []  # 用于存储当前层的节点值
            
            for _ in range(level_size):
                node = queue.popleft()  # 从队列中取出一个节点
                current_level.append(node.val)  # 将节点值加入当前层的列表
                
                if node.left:
                    queue.append(node.left)  # 如果节点有左子节点，将左子节点加入队列
                if node.right:
                    queue.append(node.right)  # 如果节点有右子节点，将右子节点加入队列
            
            result.append(current_level)  # 将当前层的节点值列表加入结果列表
        
        return result  # 返回最终的层序遍历结果
```
