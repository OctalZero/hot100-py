# 🧩 智力题 · 大厂高频脑筋急转弯

> 收录大厂面试历史上高频出现的经典智力题/脑筋急转弯，覆盖**赛马 · 试毒 · 计时 · 称重 · 概率 · 统筹 · 博弈**等主流类型。每题沿用本库风格：Mermaid 图解 + 人话思路 + 举一反三。

---

## ⚠️ 先说结论：这类题已大幅弱化（别花太多时间）

与本库「按社招考察优先级排序」的定位一致，必须诚实提醒：

- 连 **Google** 都公开表示过「面试脑筋急转弯是浪费时间，不能反映员工未来发展」。
- **2025–2026 年大厂社招，纯智力题已基本退出主流**，权重极低；偶尔出现，多为**个别面试官个人偏好**，不影响过不过。
- **本目录定位 = 图心安 + 理解思路 + 应对偶遇**，而非核心考点。

> 🔑 真正的社招核心是：**算法（Hot100）+ 系统设计 + 线上故障排查场景题 + 项目深度盘问**。智力题排在最末，临场不慌即可。

## 那为什么还是收录？

1. **字节 / 腾讯个别部门仍有偏好**——尤其是「赛马」「试毒」这类**有算法内核**的题（信息论 / 二分），思路可迁移到真算法题。
2. **部分智力题本质就是算法**：扔鸡蛋 = 动态规划；Nim 博弈 = 博弈论；这些和 Hot100 是连通的。
3. **临场不慌**——遇到至少见过套路，能在 2 分钟内给出方向，而不是冷场。

## 🚀 复习建议

1. **时间分配**：智力题花 **1–2 小时**扫一遍思路即可，**不要刷**——把时间还给 Hot100 和系统设计。
2. **重思路轻答案**：面试官看重的是「你如何把模糊问题拆解」，而不是背出数字。每题练**口述推理过程**。
3. **识别算法内核**：遇到试毒/称球，主动往「信息量 / log」上靠；遇到扔鸡蛋/Nim，主动往「DP / 博弈」上靠——把智力题「翻译」成你熟悉的算法语言，是最大的加分项。
4. **坦诚**：如果真没思路，别硬编。说「这类题我想到 XX 方向，但不确定最优解」，比胡说强。
